Instagram 팔로워 관리

사용방법

  1. 인스타그램 정보 요청 사이트(클릭)에 접속
  2. 로그인 안 되어 있으면 하기
  3. "정보 다운로드 또는 전송" 클릭
  4. "일부 정보" 클릭
  5. 쭉 내려가다 "팔로워 및 팔로잉"에 체크 후 "다음" 클릭
  6. "기기에 다운로드" 클릭
  7. 기간 → "전체 기간" 선택 후 저장,
    형식 → "JSON" 선택 후 저장,
    미디어 품질 → "낮음" 선택 후 저장
  8. "파일 만들기" 클릭

    9. (10분 정도 기다리기...)

  9. 다시 인스타그램 정보 요청 사이트(클릭)에 접속
  10. "다운로드" 클릭
  11. 비밀번호 입력하기

(삼성 브라우저) 아래 버튼 클릭 → 내 파일 → instagram~.zip 선택 후 "완료"
